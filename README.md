## 我喜欢做的东西

- **可展示的页面和小应用**：作品集、天气应用、电影库、文档站、组件示例。
- **有画面感的交互实验**：Canvas、D3、Three.js、图片处理、小游戏、可视化 Demo。
- **能讲清楚的技术沉淀**：把 Demo、踩坑、工作流和学习路线整理成 README 与文章。
- **可运行的全栈练习**：Node.js、Python、WebSocket、WebRTC、文件服务和部署实践。
